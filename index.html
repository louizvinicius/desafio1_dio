Estrutura básica do HTML

(voltar)

Pra começar, vamos deixar claro que html não é uma linguagem de programação; e sim uma liguagem de marcação usada para definir a estrutura do conteudo.
Por exemplo, o conteúdo pode ser estruturado em parágrafos, em uma lista com marcadores ou usando imagens e tabelas.
HTML consiste de uma série de elementos, que você usa para delimitar ou agrupar diferentes partes do conteúdo para que ele apareça ou atue de determinada maneira.
As tags anexas podem transformar uma palavra ou imagem num hiperlink, pode colocar palavras em itálico, pode aumentar ou diminuir a fonte e assim por diante.



Falando sobre Tags

(voltar)

O conhecimento das tags é inevitável para programadores que desejam programar em front-end, pois elas são a base da estrutura de qualquer página web.
O HTML é composto por várias tags que definem diferentes tipos de conteúdo, incluindo cabeçalhos, parágrafos, listas, imagens, links, formulários e muitos outros.
Cada tag é como uma instrução para o navegador sobre como exibir diferentes partes do conteúdo. Por exemplo, a tag " < p >" é como uma pequena etiqueta que diz ao navegador: "Aqui começa um parágrafo".
É como uma etiqueta de índice em um livro, direcionando o navegador para onde encontrar e como apresentar cada elemento na página.
Então, quando você abre uma tag "< p >", o navegador entende que deve começar a exibir o texto como um parágrafo.

Elementos HTML

(voltar)

Cada elemento é definido por tags. Por exemplo, para criar um parágrafo, usamos a tag "< p > texto desejado < /p >"

Os Principais elementos HTML:

  • "< html >"

    • Essa tag representa a base do codigo html, é a partir dela que o codigo se inicia e com isso todas as outras tags serão descendentes dela.
    A tag "< /html >" marca seu fim.

  • "< head >"

    • Essa tag fornece informações gerais (metadados) sobre o documento, incluindo o título e links para scripts e folhas de estilo.
    A tag "< /head >" marca seu fim.

  • "< link >"

    • Essa tag define relações entre o documento atual e recursos externos. Como paginas de estilo.
    Essa tag é o primeiro exemplo de tag "Simples" que são aquelas que não precisam ser fechadas.

  • "< meta >"

    • Define metadados que não podem ser especificados por outros elementos HTML, como informações sobre estilos, scripts e dados auxiliares.
    Um exempro: "< meta charset="utf-8" >"

  • "style"

    • Contém informações de estilo para um documento ou parte dele.

  • "< title >"

    • Define o título do documento, exibido na barra de título do navegador ou na aba da página.


Além disso, temos o elemento < body >, que representa o conteúdo principal do documento HTML. Cada documento pode ter apenas um elemento < body > & < /body >.



Atributos básicos

(voltar)

Os atributos básicos em HTML são informações adicionais que podem ser incluídas em tags para fornecer mais detalhes sobre um elemento.
Eles ajudam a definir o comportamento, estilo ou outras características de um elemento HTML. Por exemplo, o atributo "src" em uma tag de imagem (< img src="..." >) especifica a localização da imagem a ser exibida.
Outro exemplo é o atributo "href" em uma tag de link (< a href="...">), que define o destino do link. Esses atributos básicos são essenciais para personalizar e controlar o conteúdo em uma página da web.
